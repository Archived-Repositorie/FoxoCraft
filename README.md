## Compilation

1. Install `git`
2. Choose place for repository and run `git clone https://github.com/JustFoxx/FoxoCraft.git`
3. Go to folder called `FoxoCraft` in folder of your choice
4. Run `./gradlew`
5. Go to `./build/libs` and here you can find binaries of mod
